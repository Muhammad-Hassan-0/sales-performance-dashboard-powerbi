# AdventureWorks Executive Sales Analytics Dashboard

## Overview

This project is an interactive Power BI dashboard developed using the AdventureWorks Sales dataset. It provides comprehensive insights into sales performance, product profitability, territory analysis, and executive KPIs.

---

## Problem Statement

AdventureWorks lacked a centralized reporting solution to monitor sales performance across products, territories, and salespersons. The objective of this project was to design an interactive dashboard that enables business users to monitor KPIs and make data-driven decisions.

---

## Features

- Executive Dashboard
- Product Performance Analysis
- Sales & Territory Analysis
- Interactive Slicers
- Dynamic KPIs
- Time Intelligence
- Star Schema Data Model
- Conditional Formatting
